# Chat App

This is a simple React Native chat app prototype built using Expo. It allows users to enter a name, choose a background color, and navigate to a chat screen that reflects their selected settings.

## Features

- User name input
- Background image on start screen
- Background color selection using `TouchableOpacity`
- Navigation with React Navigation
- Color and name passed via route parameters
- Custom navigation header
- Tested on Android Emulator

## Screens

### Start Screen
- User can enter their name
- Select one of four background colors
- Button navigates to chat screen

### Chat Screen
- Displays chosen background color
- Shows user’s name in navigation bar

## Tech Stack

- React Native
- Expo
- React Navigation
